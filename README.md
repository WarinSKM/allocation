# Salmon Allocation System

A salmon inventory allocation dashboard built with React, TypeScript, and Vite.

## Clone or Download

**Clone via SSH**
```bash
git clone git@github.com:WarinSKM/allocation.git
```

**Clone via HTTPS**
```bash
git clone https://github.com/WarinSKM/allocation.git
```

**Download ZIP**

Go to [github.com/WarinSKM/allocation](https://github.com/WarinSKM/allocation), click **Code → Download ZIP**, then extract the archive.

## Install & Run

Requires **Node.js 18+** and **npm**.

```bash
cd allocation
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Other Commands

```bash
npm run build   # production build
npm run lint    # ESLint
npm run mock    # regenerate mock data
```
